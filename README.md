# EPI Lab Website - XUPT

这是西安邮电大学 EPI 实验室的官方网站项目。

## 技术栈

- Vue 3
- Vue Router
- Element Plus
- Vite
- Sass

## 功能特点

- 响应式设计
- 实验室介绍
- 研究方向展示
- 团队成员展示
- 招聘申请系统

## 开发环境设置

1. 克隆项目
```bash
git clone https://github.com/YourUsername/EPI_Labwebsite_XUPT.git
cd EPI_Labwebsite_XUPT
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```

## 项目结构

```
src/
├── components/     # 通用组件
├── views/          # 页面视图
├── router/         # 路由配置
├── App.vue         # 根组件
└── main.js         # 入口文件
```
